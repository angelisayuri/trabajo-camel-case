CAMELCASE Y LAS METODOLOGIAS DE BEM

las metodologías de nombrado BEM y camelCase mientras aplican los conocimientos básicos de HTML y CSS aprendidos en clase (estructura web, propiedades de texto, listas, margin, padding, border).

¿QUE ES BEM?

es un enfoque basado en componentes para el desarrollo web. su idea es dividir la interfaz de usuario en bloques independientes.

VENTAJAS DE BEM una lista Ordenada (ol) muestra elementos de un orden especifico, numerados automaticamente

  1. primer elemento claridad y legibilidad
  2. segundo elemento modularidad y reutilizacion
  3. Tercer elemento facilita mantenimiento
  4. Cuarto elemento evita conflictos de estilos
  5. Quinto elemento escalabilidad en proyectos grandes
